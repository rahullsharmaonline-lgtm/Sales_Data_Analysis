# 📊 Sales Data Analysis (Excel Project)

## 📌 Overview
This project focuses on analyzing sales data using Microsoft Excel. It includes data cleaning, transformation, pivot table analysis, and dashboard visualization to extract meaningful business insights.

## 🚀 Key Features
- Data cleaning and preprocessing
- Sales analysis by region and category
- Monthly sales trend analysis
- Interactive dashboard with charts (bar, pie, line)

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization (Charts)

## 📈 Analysis Performed
- Region-wise sales performance
- Category-wise sales distribution
- Monthly sales trend analysis

## 💡 Insights
- Identified top-performing regions based on total sales
- Analyzed category contribution to overall revenue
- Observed trends in sales across different months

## 📁 Files Included
- `dashboard_with_charts.xlsx` → Final dashboard with charts
- `analysis_data.xlsx` → Cleaned and structured data

## 🎯 Outcome
Built a complete Excel-based data analysis solution demonstrating data cleaning, analysis, and visualization skills suitable for entry-level data analyst roles.
